# Red Light Green Light (RLGL)  

# Overview
This project implements a Red Light Green Light (RLGL) game using OpenCV and a webcam.
Players must move during GREEN phases and remain still during RED phases. Motion detection is performed frame-to-frame, and a state machine enforces the game rules.

# Features
- Real-time webcam input
- Motion detection via frame differencing
- State machine with GREEN, RED, WARNING, DEAD
- Randomized phase durations
- Idle detection during GREEN
- Grace period during RED

# Requirements
- Python 3.8+
- OpenCV (opencv-python)
- NumPy
